Padding:

Es la distancia desde el borde de un elemento hasta su contenido. Esta propiedad es imprescindible si la caja tiene color de fondo o borde.

Si no lo tiene, se vería igual usando margin para generar los espacios. Hay que tener en cuenta que el padding por defecto aumenta el tamaño de una caja, para que esto no suceda, se usa box-sizing: border-box.

Esta propiedad se suele definir en una regla con selector universal para que todos los elementos del sitio se comporten igual. El código es:

* {box-sizing: border-box}
